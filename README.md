匿名凶案
==

《匿名凶案》设计文档：[https://docs.qq.com/doc/DS1BWZFNJTU9CeVhx](https://docs.qq.com/doc/DS1BWZFNJTU9CeVhx)

ksm 是本项目使用的 DSL ，用于编写游戏脚本，如对白等。我为 ksm 编写了一个简单的 vsc 插件，集成了基本的语法服务和编译功能：[https://github.com/Heaveeeen/krill-script-marco-for-vsc-extension](https://github.com/Heaveeeen/krill-script-marco-for-vsc-extension)

本仓库不应该公开，因为其中有一些版权比较灰色的内容。如果要将这些素材用到其他地方，请自行承担侵权风险。

`ksm/dist/`中的内容暂不提交。
